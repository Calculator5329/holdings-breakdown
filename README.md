# Holdings Breakdown

Analyze and categorize your investment portfolio across asset classes and investment types.

## What It Is

Holdings Breakdown processes multiple CSV files containing your investment holdings and automatically categorizes them into stocks, bonds, cash, market funds, and leveraged products. The app generates an interactive dashboard showing your portfolio composition by category, with downloadable CSV exports for further analysis. Perfect for understanding asset allocation and rebalancing needs.

## Tech Stack

- React 19 with TypeScript
- Vite for fast development
- CSS for styling

## Getting Started

```
npm install
npm run dev
```

Place your CSV holding files in the `public` folder and the app will automatically load and process them on startup. Download the categorized breakdowns directly from the dashboard.
